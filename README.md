# Landing Page — Photographer

Одностраничный лендинг для профессионального фотографа с акцентом на визуальную подачу, типографику и пользовательские взаимодействия.

Проект выполнен как фронтенд-реализация по дизайн-макету (Figma) и демонстрирует навыки верстки, работы с layout, интерактивными элементами и структурированием страницы.


## Функциональность

- Навигация по странице с якорными ссылками
- Скролл между секциями
- Hero-секция с CTA
- Блок «Обо мне»
- Портфолио с сеткой изображений
- Блок цен
- FAQ с раскрывающимися ответами (HTML + CSS)
- Footer с навигацией и социальными иконками

## Используемые технологии

- HTML5 (семантическая разметка)
- CSS3  
  - Flexbox  
  - Grid  
  - Transitions  
- SVG-иконки
- Подключение кастомных шрифтов
- Верстка по дизайн-макету Figma

## Как запустить проект

1. Склонировать репозиторий
2. Открыть файл `index.html` в браузере


## Цель проекта

- Практика чистой верстки
- Работа с фиксированными и адаптивными layout
- Реализация интерактивных элементов без JavaScript
- Подготовка проекта для портфолио
